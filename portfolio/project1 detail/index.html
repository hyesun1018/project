Date

Client

TOCOBO

overview

친환경 에너지 사업을 선도하는
기업의 위상과 가치에 대응하는
홈페이지로 리뉴얼

TOCOBO가 지향하는 젊은 에너지와 트렌디함을 강조하기 위해
충분한 여백과 정형화 되지 않은 그리드를 레이아웃을 사용하였으며,
실제 사용되는 제품의 스킨케어 스타일을 이용한 TOCOBO의 아이덴티티를 강조하였습니다.

Project Goal

청량감이 느껴지는
TOCOBO 감성

TOCOBO의 비피다, 코코넛, 락토코쿠스 용해물 재료를
모티브로 제작하였으며 하나하나 디테일에 신경 쓴 일러스트와 모션이 TOCOBO의 감성을 느끼게 해줍니다.

UI/UX Concept

사용자 환경에
맞춰 최적화되는
반응형웹

다양한 디바이스와 해상도에 맞게 최적화되는 반응형웹으로 개발하여 여러 환경의 사용자에게 동일한 사용자경험을 제공하며, 사이트의 유지관리 효율성을 높였고 검색엔진에 친화적이게 제작되었습니다.
